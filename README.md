# Nekote

## 概要

猫の手も借りたい自分が今後の .NET ベースの開発を効率化するための C# クラスライブラリーです。

基本的には自分や自分と関わりのある会社のためのものですが、1) それぞれの機能をできるだけ独立的かつカスタマイズ性が高いように作る、2) 他者のプロジェクトの既存の仕様やコードに影響しにくい、を念頭に開発するので、特定の機能のみ API 的に使っていただくと便利なこともあるでしょう。

しばらくは、そのときに開発中のソフトに必要な機能を順次追加していくことになります。ソフト側で中規模のものがいくつか落ち着く頃、Nekote はたぶん 0.0.100 くらいになっています。それくらいで最初の正式版である 0.1.0 をリリースし、それからはドキュメントの整備にも取り掛かる予定です。
